# Architecture Decision Records

This directory holds architectural decisions for Precept in lightweight [MADR](https://adr.github.io/madr/) form. New decisions get a new numbered file; existing decisions are superseded rather than rewritten.

| ID | Title | Status | Related tickets |

## Writing a new ADR

1. Copy the latest ADR as a template.
2. Number sequentially (`0002-…`, `0003-…`).
3. Set Status to `Proposed`, `Accepted`, `Deprecated`, or `Superseded by NNNN`.
4. Keep it short: Context, Decision, Consequences, Non-goals. One page is usually enough.
5. Link the ticket(s) the ADR addresses.
6. Update this index.
